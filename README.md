# Phone Accessories Shop Website

This is a simple and professional phone accessories website built using **HTML**, **CSS**, and **VS Code**.  
It includes a logo, homepage layout, and sections for products.

## Features
- Custom logo included on the website
- Clean and responsive layout
- Product sections for phone accessories
- Easy to edit and update from VS Code

## Technologies Used
- HTML
- CSS
- VS Code

## How to Use
1. Download or clone the project to your computer.
2. Open the project folder in **VS Code**.
3. Open `index.html` and click **"Run"** or open it in your browser.
4. Edit the HTML and CSS files to customize the website.
